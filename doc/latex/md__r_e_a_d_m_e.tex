Over the course of this project we designed and implemented Penn\+OS, a simulated os that can handle many aspects of OS functionality. This includes process management, file system interactions (Penn\+FAT), and basic shell operations.\hypertarget{md__r_e_a_d_m_e_autotoc_md12}{}\doxysubsubsection{\texorpdfstring{Key accomplishments\+:}{Key accomplishments\+:}}\label{md__r_e_a_d_m_e_autotoc_md12}

\begin{DoxyItemize}
\item {\bfseries{Penn\+FAT Implementation}}\+: Developed a fully functional file allocation table (FAT) based file system (Penn\+FAT). Penn\+FAT supports file creation, deletion, reading, and writing, along with permission handling, and other expected functionality of a file system.
\item {\bfseries{Kernel Implementation}}\+: Kernel capabilities to manage processes with PCB and PID management systems. This enables process scheduling and management.
\item {\bfseries{Shell Functionality}}\+: Built a shell that allows for process interaction, file system navigation, and testing with custom-\/built commands and scripts.
\item {\bfseries{Testing Framework}}\+: Developed testing suites for both k-\/level and u-\/level functions to make sure the OS components functioned properly.
\end{DoxyItemize}

This project gave us a solid foundation of operating systems.\hypertarget{md__r_e_a_d_m_e_autotoc_md13}{}\doxysubsection{\texorpdfstring{Description of Code and Code Layout}{Description of Code and Code Layout}}\label{md__r_e_a_d_m_e_autotoc_md13}
